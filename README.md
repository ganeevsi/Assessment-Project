# Github Finder

Searches Github.com user profiles and displays Profile Info,Latest Repositories,Followers
and Following of the searched user profile.

### Version
1.0.0

## Usage

Open index.html


Technologies Used
1.HTML
2.CSS
3.JavaScript
4.JQuery
5.Bootstrap

API Used :-
Github API

API Endpoints
1.Display Profile
2.Display Repos
3.Followers
4.Following
